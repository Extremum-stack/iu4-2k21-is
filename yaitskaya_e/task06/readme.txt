task06.exe.idb - файл для IDA с комментариями

В ходе разбора программы было выявлено, что она требует на вход два обязательных агумента:
login и password
Они могут быль любыми.
Программа выводит связный список из аргументов командной строки, начиная с третьего argv[3]
Если одни из этих аргументов является строкой "TEST", то выводится сторока "That's fine!"
