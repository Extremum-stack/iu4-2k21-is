На вход программы поступают аргументы из командной строки. 0-ой аргумент - имя программы, 1-ый - имя пользователя, 2-ой - пароль.
Из последующих аргументов формируется двухсвязаный список. Формирование списка происходит в функции "process_args". 
Внутри этой функции в функции "list_create" происходит инициализация списка (head = 0, tail = 0). Затем в цикле от i = 0 до i < argc - 3
происходит добавление node в ранее инициализированный список. Добавление node происходит при помощи функции list_add, в которой вызывается 
функция node_create. После это в main вызывается функция "show_list", которая выводит список в консоль.
После этого происходит вызов функции "check_list_for_TEST", которая проходит по списку и проверяет есть ли в списке node со строкой "TEST".
Если такая node есть, то в консоль выводится фраза "That's fine!". После этого происходит высвобождение памяти из-под списка с помощью функции "list_free".
